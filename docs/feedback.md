# Feedback and Bug Reports

Use this page to report bugs, documentation issues, and improvement requests for BAM addons.

The form opens GitHub Issues with a prefilled title and report body. You can review the issue before submitting it on GitHub.

If you do not have a GitHub account, use the email button to send the same report directly.

File attachments are added after the issue page opens. Drag and drop screenshots, videos, or `.blend` files into the GitHub issue editor before submitting.
