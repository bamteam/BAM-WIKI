# Installation

## Requirements

- Blender 4.5 or newer.
- Internet access for Blender Extensions Store installation, or a BAM Decal IT addon archive for manual installation.
- Permission to install Blender addons on your machine.

The addon was developed and tested on Blender 5.0.

## Method 1: Blender Extensions Store

Use this method first when BAM Decal IT is available through Blender Extensions.

Official extension pages:

- [BAM](https://extensions.blender.org/add-ons/bam/)
- [BAM Decal IT](https://extensions.blender.org/add-ons/bam-decalit/)

1. Open Blender.
2. Go to `Edit > Preferences > Get Extensions`.
3. Search for `BAM`.
4. Install the base BAM extension if it is not installed yet.
5. Search for `BAM Decal IT`.
6. Click `Install`.
7. Enable the extension if Blender does not enable it automatically.
8. Restart Blender if the BAM Decal IT panel does not appear immediately.

## Method 2: Manual Add-on Install

Use this fallback method when you have a local `.zip` archive or addon folder.

1. Open Blender.
2. Go to `Edit > Preferences > Add-ons`.
3. Click the install button.
4. Select the BAM Decal IT `.zip` file.
5. Enable the addon checkbox after Blender finishes installing it.

## Location

After installation, open:

```text
View3D Sidebar > [BAM] Decalit
```

The panel contains the Decal Library, preview controls, placement buttons, and projection settings.

## First Check

Open the panel and confirm that the Decal Library area loads without errors. If the addon does not appear, restart Blender and check that the addon is enabled in preferences.
