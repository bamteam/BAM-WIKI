# FAQ

## What Blender version should I use?

BAM Decal IT should work with Blender 4.5 or newer. It was developed and tested on Blender 5.0.

## Where is the addon located?

Open `View3D Sidebar > [BAM] Decalit`.

## Which projection mode should I start with?

Target Topology is the default mode and the best starting point for most surface placement. Volume Mode is for decals that need to keep raised mesh volume. Simple Mode uses the decal topology as-is and adds Subdiv for projection detail.

## How do I report a bug?

Use the [Feedback page](/feedback) to open a prefilled GitHub Issue for BAM Decal IT, or go directly to the [BAM-WIKI issues page](https://github.com/bamteam/BAM-WIKI/issues).
