# Installation

## All Addons

Use this page for the common BAM addon installation flow. Install from the Blender Extensions Store when possible, and use manual addon installation only when you have a local archive.

## Requirements

- A Blender version supported by the addon you install.
- BAM Automat: Blender 4.2 or newer, based on the current extension manifest.
- BAM Decal IT: Blender 4.5 or newer.
- Internet access for Blender Extensions Store installation, or a local addon archive for manual installation.
- Permission to install Blender addons on your machine.

## Method 1: Blender Extensions Store

Use this method first for BAM addons distributed through Blender Extensions.

Official extension pages:

- [BAM](https://extensions.blender.org/add-ons/bam/)
- [BAM Decal IT](https://extensions.blender.org/add-ons/bam-decalit/)

1. Open Blender.
2. Go to `Edit > Preferences > Get Extensions`.
3. Search for `BAM`.
4. Install the base BAM extension if it is not installed yet.
5. Search for the BAM addon you want to install.
6. Click `Install`.
7. Enable the extension if Blender does not enable it automatically.
8. Restart Blender if the BAM panel does not appear immediately.

## Method 2: Manual Add-on Install

Use this fallback method when you have a local `.zip` archive or addon folder.

1. Open Blender.
2. Go to `Edit > Preferences > Add-ons`.
3. Click the install button.
4. Select the addon `.zip` file.
5. Enable the addon checkbox after Blender finishes installing it.

## First Check

Open the BAM panel in Blender and confirm that the addon UI loads without errors. If the addon does not appear, restart Blender and check that it is enabled in preferences.
